# CovJSON NdArray

The actual data values in a flat 1D array with shape metadata to reconstruct the N-dimensional grid.

## Finland VAP NdArray
```json
{
  "type": "NdArray",
  "dataType": "integer",
  "axisNames": ["t", "y", "x"],
  "shape": [18, 200, 320],
  "values": [106, 95, 111, ...]
}
```
18 years × 200 rows × 320 columns = 1,152,000 values.
